# Security

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, report them privately through one of the following channels:

- Email: <secure@microsoft.com>
- The Microsoft Security Response Center (MSRC) at
  <https://msrc.microsoft.com/create-report>

Microsoft follows the principle of
[Coordinated Vulnerability Disclosure](https://aka.ms/opensource/security/cvd).

Please include:

- A description of the issue and its potential impact.
- Steps to reproduce or a proof-of-concept.
- The Fabric capacity SKU, region, and any specific item (notebook, pipeline,
  data agent, KQL DB) involved.

You should receive an acknowledgement within 24&nbsp;business hours. If you do
not, please follow up to ensure the original message was received.

## Scope

This Jumpstart deploys synthetic, non-PHI data and does not handle real
patient information. If you discover an issue in dependencies (Microsoft
Fabric platform, Power BI, Real-Time Intelligence), please report directly to
MSRC; this repository only owns the demo notebooks, pipelines, and report
definitions.
